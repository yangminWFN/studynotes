# 堆排序

```java
	public static void heap_sort(int[] a) {
        int n=a.length;
        build_heap(a,n);
        for(int i=n-1;i>=0;i--){
            //交换堆顶元素a[0]和最后一个元素a[i]
            swap(a,i,0);
            //从堆顶进行调整，使其保持大顶堆结构；此时大顶堆中元素个数为i个
            heapify(a,i,0);
        }
    }

    //从最后一个非叶子节点开始构建初始堆
    public static void build_heap(int[] a, int n) {
        int last_node = n - 1;
        int last_parent = (last_node - 1) / 2;
        for (int i = last_parent; i >= 0; i--) {
            heapify(a, n, i);
        }
    }

    /**
     * 在含有n个元素的数组中，从节点 i 向下递归调整堆结构，使其保持为大顶堆
     * @param tree 当前数组
     * @param n 数组元素个数
     * @param i 调整堆结构的起始节点下标
     */
    public static void heapify(int[] tree, int n, int i) {
        if (i >= n) return;
        int left = i * 2 + 1;
        int right = i * 2 + 2;
        int max = i;
        if (left < n && tree[left] > tree[max]) {
            max = left;
        }
        if (right < n && tree[right] > tree[max]) {
            max = right;
        }
        if (max != i) {
            swap(tree, i, max);
            heapify(tree, n, max);
        }
    }

    private static void swap(int[] nums, int i, int j) {
        int temp = nums[i];
        nums[i] = nums[j];
        nums[j] = temp;
    }
```

## 思路

### 什么叫大顶堆？

满足两个条件：

1. **是一颗完全二叉树**
2. **父节点值大于等于左右节点值**

### 如何构造一个大顶堆？

两种方式：

1. **原数组已是大顶堆结构，其中一个节点改变，需要从该节点开始递归向下调整为大顶堆结构**：比较该节点与左右孩子节点的值，选择三个节点中值最大的节点，记录最大值下标，如果该下标不是父节点，则再递归地从该下标进行结构调整。**三个参数：tree, n,i** 
2. **原数组本不是大顶堆结构，需要构建大顶堆**：取得（数组代表的）二叉树中最后一个节点的父节点last_parent，从该父节点开始,依次递减地调整大顶堆结构，即可将整个二叉树构成一个大顶堆



```java
//构建一个大顶堆类（不带自动扩容）
class MaxHeap {
    private int[] data; // 下标从 1 开始
    private int capacity; // 容量
    private int count; // 实际元素个数

    // 构建一个特定容量的大顶堆
    public MaxHeap(int capacity) {
        this.capacity = capacity;
        this.data = new int[capacity + 1];
        this.count = 0;
    }

    // 判断堆是否为空
    public boolean isEmpty() {
        return count == 0;
    }

    // 往堆中添加元素
    public void offer(int item) {
        assert (count < capacity);
        count++;
        // 插入到数组最后
        data[count] = item;
        // 使用上浮操作调整堆结构
        shiftUp(count);
    }

    // 下标为 k的元素进行上浮
    private void shiftUp(int k) {
        int parent = k / 2;
        while (parent > 0 && data[parent] < data[k]) {
            swap(data, parent, k);
            k = parent;
            parent = k / 2;
        }
    }

    // 弹出堆顶元素
    public int poll() {
        assert (count > 0);
        int root = data[1];
        // 交换堆顶元素和最后一个元素
        swap(data, count, 1);
        count--;
        // 使用下沉操作调整堆
        shiftDown(1);
        return root;
    }

    // 对下标为 k 的元素进行下沉操作
    private void shiftDown(int k) {
        int j = k * 2;
        while (j <= count) {
            if (j + 1 <= count && data[j + 1] > data[j]) {
                j = j + 1;
            }
            if (data[j] > data[k]) {
                swap(data, j, k);
                k = j;
                j = k * 2;
            } else
                break;
        }
    }

    private void swap(int[] a, int i, int j) {
        int t = a[i];
        a[i] = a[j];
        a[j] = t;
    }
}

public class MaxHeapTest {
    public static void main(String[] args) {
		//使用一个大顶堆来进行堆排序
        int[] a = new int[] { 4, 2, 1, 3, 6, 8, 7, 9, 5, 0 };
        MaxHeap maxHeap = new MaxHeap(a.length);
        for (int i = 0; i <= a.length - 1; i++) {
            maxHeap.offer(a[i]);
        }
        int i = a.length - 1;
        while (!maxHeap.isEmpty()) {
            a[i--] = maxHeap.poll();
        }
        System.out.println(Arrays.toString(a));
    }
}
```

```java
public class HeapSortTest{
    
    public static void heap_sort(int[] a) {
        int n = a.length;
        heapify(a, n);
        for (int i = n - 1; i > 0; i--) {
            swap(a, i, 0);
            shiftDown(a, i, 0);
        }
    }
    
    //从最后一个非叶子节点开始往前做下沉操作，将数组转化为堆结构
    public static void heapify(int[] a, int n) {
        int parent = (n - 1) / 2;
        for (int i = parent; i >= 0; i--) {
            shiftDown(a, n, i);
        }
    }
    
    //将下标为 k 的元素下沉
    public static void shiftDown(int[] a, int n, int k) {
        int j = k * 2 + 1;
        while (j < n) {
            if (j + 1 < n && a[j + 1] > a[j]) {
                j = j + 1;
            }
            if (a[j] > a[k]) {
                swap(a, j, k);
                k = j;
                j = 2 * k + 1;
            } else
                break;
        }
    }

    public static void swap(int[] a, int i, int j) {
        int t = a[i];
        a[i] = a[j];
        a[j] = t;
    }   
}
```
