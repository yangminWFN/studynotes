# [25. K 个一组翻转链表](https://leetcode-cn.com/problems/reverse-nodes-in-k-group/)

```java
/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode() {}
 *     ListNode(int val) { this.val = val; }
 *     ListNode(int val, ListNode next) { this.val = val; this.next = next; }
 * }
 */
class Solution {
    public ListNode reverseKGroup(ListNode head, int k) {
        //K个一组反转链表
        ListNode dummy = new ListNode(-1);
        //指向已反转链表的最后一个节点
        ListNode cur = dummy;
        //指向正在反转链表的第一个节点
        ListNode p = null;
        //指向正在反转链表的最后一个节点，用于切断前面链表与后面链表
        ListNode tail = null;
        while(head != null){
            p = head;
            //让 head 指向下一组未反转链表的第一个节点,tail指向当前正在反转链表的最后一个节点
            for(int i = 0;i< k;i++){
                if(head != null){
                    if(i == k-1){
                        tail=head;
                    }
                    head = head.next;
                }else{
                    //如果最后一组元素个数小于 k ，那么直接将最后一组接在后面即可
                    cur.next = p;
                    return dummy.next;
                }
            }
            //断开前面链表与后面的链表
            tail.next=null;
            cur.next = reverseList(p);
            while(cur.next != null){
                cur = cur.next;
            }
        }
        return dummy.next;
    }
    public ListNode reverseList(ListNode head){
        ListNode pre = null;
        ListNode cur = head;
        ListNode p = null;
        while(cur != null){
            p = cur.next;
            cur.next = pre;
            pre = cur;
            cur = p;
        }
        return pre;
    }
}
```

## 思想

**使用一个dummy节点来指向已反转的链表的头节点，cur节点指向已反转链表的最后一个节点（用于连接继续反转好的子链表）；指针 p 指向当前正在反转链表的第一个节点，指针 tail 指向当前正在反转链表的最后一个节点 ，指针head指向后续要反转链表的第一个节点，若反转完后head为null，返回 dummy.next ;** 

**其中tail节点主要是为了切断当前反转的链表与后续链表之间的关系；如果最后一组链表没有k个元素，则直接接在cur指针后面并返回。**

```java
    //K个一组反转链表
    public ListNode reverseKGroup(ListNode head, int k) {
        int n = 0;
        ListNode p = head;
        while(p != null){
            n++;
            p = p.next;
        }
        ListNode dummy = new ListNode(-1,head);
        ListNode pre = dummy;
        ListNode cur = dummy.next;
        ListNode remain = null;
        int curLen = 0;
        while(cur != null ){
            //如果不是最后一组小于k个元素的情况
            if((curLen + k) <= n){
                remain = cut(cur, k);
                pre.next = reverseList(cur);
                while(pre.next != null){
                    pre = pre.next;
                }
                cur = remain;
                curLen += k;
            }else{
                pre.next = cur;
                break;
            }
        }
        return dummy.next;

    }

    //从头节点为 head 的链表里切前n个节点，返回剩余链表的头节点
    public ListNode cut(ListNode head,int n){
        if(head == null) return head;
        ListNode p = head;
        while(p.next != null && (--n) != 0){
            p = p.next;
        }
        ListNode res = p.next;
        p.next = null;
        return res;
    }

    public ListNode reverseList(ListNode head){
        ListNode pre = null;
        ListNode cur = head;
        ListNode q = null;
        while(cur != null){
            q = cur.next;
            cur.next = pre;
            pre = cur;
            cur = q;
        }
        return pre;
    }
```

这种方式的主要思路在于：==从一个头节点为`head`的链表中，切掉前 `n` 个节点，返回剩余链表的头节点，即 `cut` 方法==

==在记录下链表长度和当前长度，判断是否最后一组链表长度小于 `k`，若小于 `k` 那么直接将剩余节点接在已反转好链表的后面==
