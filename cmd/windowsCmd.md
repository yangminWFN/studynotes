# Windows常用命令



|     win+alt+f4     |        关机窗口        |
| :----------------: | :--------------------: |
|     **alt+f4**     |      **关闭应用**      |
|   **alt+space**    |    **当前窗口操作**    |
|    **win+tab**     |    **整体切换窗口**    |
|  **win+上下左右**  | **调整窗口位置和大小** |
|      **alt**       |     **聚焦工具栏**     |
|    **alt+alt**     |   **取消聚焦工具栏**   |
| **Ctrl+Shift+Esc** |   **打开任务管理器**   |
|    **ctrl+tab**    |   **浏览器切换标签**   |
|                    |                        |
