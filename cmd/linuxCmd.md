## nano编辑器

|   ctrl+O   |     保存     |
| :--------: | :----------: |
| **ctrl+X** |   **退出**   |
| **ctrl+K** | **剪切一行** |
| **ctrl+U** |   **粘贴**   |
| **ctrl+6** |   **标记**   |
| **alt+6**  |   **复制**   |

## 文件操作

chmod 777 test.txt：修改文件权限为所有人可读可写可执行
		chmod 777 mydir：修改文件目录为所有人可读可写可执行
		chmod -R 777 mydir：修改文件目录及其子目录(不包括其中文件)为所有人可读可写可执行
		7=4(可读)+2（可写）+1（可执行）
		7（此用户）7（同组用户）7（其他用户）
		-：普通文件
		d：目录
